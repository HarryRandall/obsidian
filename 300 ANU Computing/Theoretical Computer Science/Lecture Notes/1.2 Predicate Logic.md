#### Lecture 1
##### Variable Assignment
- We don't actually need all of the clauses. Some of them can be derived from others.
> **Example**
- $True$, then holds (regardless of $\sigma$)
- $p \vee q$, then it holds if $p$ is $true$ under $\sigma$ or $q$ is $true$ under $\sigma$
- $\neg p$, then it holds if $p$ is $false$ under $\sigma$

- $false$ - this is the same as $\neg True$
- $p \land q$ - this is the same as $\neg (\neg p \vee \neg q)$
- $p \implies q$ - this is the same as $\neg p \land q$

##### Expressive Completeness
A set $S$ of logical connectives is **expressively complete** if all Boolean functions can be defined using connectives in $S$.
- ($\land, \vee, \neg, \implies, True, False$) this is the base set with most of the functions, and it's expressively complete.
- ($\vee, \neg, True$) is expressively complete.
	- We can prove something like $\land$ with the formula
	- $\land = \neg (\neg p \vee \neg q)$ 
-  ($\land, \neg, True$) is expressively complete.

| p   | q   | $\neg p$ | $\neg q$ | $\neg p \vee \neg q$ | $\neg (\neg p \vee \neg q)$ | $p \land q$ |
| --- | --- | -------- | -------- | -------------------- | --------------------------- | ----------- |
| T   | T   | F        | F        | F                    | T                           | T           |
| T   | F   | F        | T        | T                    | F                           | F           |
| F   | T   | T        | F        | T                    | F                           | F           |
| F   | F   | T        | T        | T                    | F                           | F           |
- ($\vee, \neg)$ is not expressively complete.
- ($\land, True$) is not expressively complete.
