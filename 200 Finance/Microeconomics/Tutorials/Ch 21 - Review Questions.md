## Formulas & Definitions

| Term | Definition |
|------|------------|
| Cost Minimization | Choosing inputs to minimize cost for given output |
| Conditional Factor Demand | Optimal inputs as function of output and input prices |
| Cost Function | Minimum cost as function of output and input prices |
