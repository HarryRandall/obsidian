## Deviations From Return Normality
- Deviations from normality is an important issue.
- Additional risk measures to indicate the likelihood an asset would experience extreme negative returns (downside risks)

![[Pasted image 20250920145055.png]]

- If return distribution is positively skewed, $\sigma$ will overestimate risk.
- If return distrbution is negatively skewed, $\sigma$ will underestimate risk.
- If return distribution exhibits "fat tails", $\sigma$ will underestimate the likelihood of extreme gains and losses.
